# Simple Flask Web Application

This is a simple Flask web application with one route that returns "Hello from Flask!".

## How to Run Locally

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the application:
   ```
   python app.py
   ```

3. Open your browser and go to http://localhost:5000

## Deployment

This application is configured to be deployed on Render as a web service.
